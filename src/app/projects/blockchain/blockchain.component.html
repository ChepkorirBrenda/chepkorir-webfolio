"I created the business network for Truck Hunt in the Hyperledger Composer, wrote the code for its assets, participants, transactions and the functions behind the transactions. Then, I added some basic access rules in the Access Control List (ACL) file and deployed it to the Hyperledger Fabric."

"All resources belonging to the network belonged to the owner of the network (a Carrier - J. B. Hunt in this case) who had Read/Write access to them. The other participants (other carriers, and shippers) had Read only access. The network would store the hashed versions of the pdf of the BOLs with other metadata about the related shipment which would be available for viewing to all participants of the network."

"The Hyperledger Composer and Fabric were choice tools for the task seeing as the Fabric is an enterprise grade Blockchain technology. The linuxONE Virtual Machine made creating a default API for the network easier which was implemented using Swagger."
