"I used the basic calculator application to explain 'old-school' front-end web application development with just JavaScript, HTML, and CSS. I wrote the code in playcode.io where the application is also hosted. The basic weather application is hosted on Heroku and in codesandbox.io (where I also wrote the code). It uses the MetaWeather API to retrieve basic weather data for various locations."

"My goal was to explain the basics of web application development in 50 minutes, but not to undermine code quality or other related standards. Basically, the code was not of focus but rather the ideas, tools, and design of the applications. I touched on the idea of being fullstack developers who also understand the significance of cloud development and DevOps in today's job market."
